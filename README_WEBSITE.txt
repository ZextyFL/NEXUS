WEBSITE / NETLIFY INSTRUCTIONS
=============================

1. Open config.js
2. Set apiUrl to:
   http://gamenode01.flexynode.com:25798/nexus
3. Set apiKey to your chosen secret key
4. Upload these website files to a GitHub repo
5. Connect the repo to Netlify
6. Deploy

These files go on GitHub/Netlify:
- index.html
- style.css
- app.js
- config.js
- netlify.toml

Important:
The apiKey here must match FIVEM_SCRIPT/config.lua exactly.
